Paginated Table Extractor

Efficiently automates the extraction of data from paginated HTML tables. Specifically, it's a python + selenium script that accepts elements such as x-path that describe where on the page the table, next button, and similar elements are. Then, the data is automatically extracted from the paginated table into panda tables and/or excel spreadsheets as efficiently as possible.
