Honorable Mentions/Alternatives

I recommend pretty much any arch based distro (such as manjaro or just pure arch) mostly because the of the AUR. I used Arch Linux for about a year before switching to Manjaro and learned so much about Linux, so I would highly recommend giving Arch Linux a try. KDE is my favorite desktop environment and I highly recommend trying it out if you've only ever used Gnome (Ubuntu's desktop environment). I3 is a very nice window manager and is the first WM I would recommend for someone testing out the waters. VScode used to be my favorite editor, but I found that the vim emulation didn't give me as much control as I prefer. Still, its extensibility, integrated debuggers, and *crazy* features such as the ability to develop inside a docker container or through ssh is awesome. Alacritty is another GPU accelerated terminal emulator that worked great for me that I recommend for speed, and st is a fantastic minimalist terminal emulator that I used for a long time. Polybar is arguably the best status bar to use with tiling window managers and I still recommend it, but I appreciate the ease of use of bumblebee, hence why I made the switch. I actually prefer thunderbird over mailspring as an email client, but I was running into issues authenticating my university email with it, so I'm using mailspring until I graduate. Qutebrowser is a very comfortable browser for a vim user, but I'm still using Brave because of LastPass integration and better ad-blocking, plus the vimium extension gives Brave most of the comforts that Qutebrowser offers. I'll probably make a switch to using Pass instead of LastPass and setup pihole to block ads, at which point I may make a full switch to Qutebrowser.
